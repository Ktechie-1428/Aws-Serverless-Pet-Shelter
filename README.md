# AWS Serverless Pet Shelter (In Progress)

## Overview
A full-stack serverless application that allows users to view pets and submit adoption applications.  
This project is being built to demonstrate real-world cloud engineering workflows using AWS.

## Tech Stack
- AWS Lambda
- Amazon API Gateway
- Amazon DynamoDB
- AWS SAM (Serverless Application Model)
- React (Vite)
- Python

## Architecture
- Backend: Serverless microservices deployed with AWS SAM
- Frontend: React application consuming a live API Gateway endpoint
- Data: DynamoDB for pet and adoption records
